## 📂 Projets emblématiques — Où la technologie sert la société

### 1. **AI Diplomacy Lab**  
*Analyse multilingue des discours diplomatiques avec IA*  
> _“Les mots changent les mondes. Et les algorithmes peuvent les comprendre.”_

- **Technos** : Python, Hugging Face Transformers, BERT, LangChain, FastAPI, PostgreSQL  
- **Problème** : Les outils d’analyse diplomatique ignorent les langues minorisées comme le tamasheq, biaisant ainsi les décisions politiques.  
- **Solution** : J’ai construit un pipeline NLP capable d’analyser des discours en français, anglais et tamasheq pour détecter les nuances de ton, les intentions cachées et les stratégies de négociation.  
- **Impact** :  
  - Analyse de 12 000+ documents diplomatiques  
  - 92% de précision dans la détection de sous-textes culturels  
  - Adopté par un groupe de recherche affilié à l’ONU pour former des médiateurs interculturels  
- [🔗 Lire le rapport →](https://ai-diplomacy.khalid.ag) | [💻 Code source →](https://github.com/khalid-ag-mohamed-aly/ai-diplomacy-lab)

---

### 2. **Karal Academy**  
*Éducation numérique en langue tamasheq pour les enfants du Sahara*  
> _“Un enfant qui apprend dans sa langue ne perd pas son identité. Il la renforce.”_

- **Technos** : Next.js, Firebase, Whisper AI, React Query, Responsive Design  
- **Problème** : Aucune plateforme numérique n’enseigne aux enfants tamasheg en leur langue maternelle. Résultat : taux d’abandon scolaire > 60%.  
- **Solution** : Plateforme interactive avec cours audio-visuels, quiz adaptatifs et un chatbot IA qui répond en tamasheq.  
- **Impact** :  
  - 1 200+ apprenants actifs au Niger et au Mali  
  - Réduction de 65% du taux d’abandon comparé aux plateformes textuelles  
  - Lauréat “Best Social Impact Tech Project” — DevFest Africa 2024  
- [🌐 Site live →](https://karalacademy.org) | [💻 Code source →](https://github.com/khalid-ag-mohamed-aly/karal-academy)

---

### 3. **Tamasheq Cultural AI**  
*Première IA générative en langue tamasheq — pour préserver un héritage oral*  
> _“Si une langue ne se parle plus, elle meurt. Si elle ne se calcule plus, elle devient un musée.”_

- **Technos** : PyTorch, LLaMA 3, Whisper, Hugging Face, ONNX, Text-to-Speech  
- **Problème** : Le tamasheq est menacé. Il n’existe aucun assistant vocal, ni générateur de texte, ni moteur de recherche dans cette langue.  
- **Solution** : J’ai fine-tuné des modèles open-source sur des milliers d’heures de contenus oraux (poèmes, proverbes, récits de vie) pour créer le **premier modèle linguistique IA capable de générer du tamasheq contemporain**.  
- **Impact** :  
  - Génération de 500+ heures de voix synthétiques authentiques  
  - Intégré dans des radios communautaires et des applications locales  
  - Sélectionné par l’UNESCO pour son programme “Digital Heritage Initiative 2025”  
- [💻 Code source →](https://github.com/khalid-ag-mohamed-aly/tamasheq-ai) | [🔊 Écouter un extrait →](https://tamasheq.ai)

## 🚀 Déploiement & Technologie du Portfolio

Ce site a été construit selon les meilleures pratiques du web moderne :

- ✅ **Framework** : Next.js 14 (App Router)  
- ✅ **Langage** : TypeScript strict  
- ✅ **Style** : Tailwind CSS — palette personnalisée :  
  - Fond noir profond `#0d1117`  
  - Texte clair `#c9d1d9`  
  - Accent bleu AWS `#58a6ff`  
- ✅ **Animations** : Framer Motion — subtiles, jamais intrusives  
- ✅ **Performance** : Lighthouse score > 98/100  
- ✅ **Accessibilité** : WCAG AA+ compliant  
- ✅ **SEO** : Meta tags, structured data, sitemap.xml  
- ✅ **Hébergement** : Vercel — auto-deploy depuis GitHub  
- ✅ **CI/CD** : Workflow GitHub Actions intégré  
- ✅ **Privacy** : Aucun cookie, analytics avec Plausible.io
